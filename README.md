# Student Management System (JDBC + PostgreSQL)

## 🚀 Features
- Add Student
- View Students
- Update Marks
- Delete Student

## 🛠 Tech Stack
- Java
- JDBC
- PostgreSQL

## ▶️ How to Run
1. Setup PostgreSQL database
2. Update DB credentials in DBConnection.java
3. Run Main.java
